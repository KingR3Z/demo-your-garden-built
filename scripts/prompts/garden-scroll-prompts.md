### END FRAME — Exploded Garden (16:9, 2K quality)
Use the start frame as reference image, then prompt:
```
Same luxury garden from the same angle, but now showing an exploded/deconstructed view. All elements are separated and floating in space: paving stones lifted and spread apart, plants hovering above their beds, the pathway stones scattered upward, furniture pieces separated, lighting fixtures floating individually. Each component is clearly visible with space between them. Maintain the same clean white background, same lighting, same style. No text. The explosion should be dramatic but organized — elements spread evenly in all directions including vertically. 2048x1152 resolution.
```

## For Kling 3.0 (Video Generation)

### Assembled → Exploded Transition
```
Smooth continuous transition where the complete garden gradually deconstructs. Paving stones gently lift and separate, plants float upward from their beds, pathway stones scatter in slow motion, furniture pieces drift apart revealing their construction. The movement should be very slow, elegant, and controlled — like a luxury product teardown video. Camera stays perfectly still. White background throughout. No text. Duration: 5-7 seconds.
```

Slow cinematic drone shot sweeping over a lush English country garden at golden hour. Warm afternoon light filters through mature oak trees, casting long shadows. The camera rises gently to reveal a stunning patio area with outdoor dining setup overlooking rolling countryside. Photorealistic, 4K quality, shallow depth of field. Duration: 5-8 seconds.

As darkness falls, garden pathway lights, tree uplights, and patio string lights gradually illuminate, transforming the space into an enchanting evening retreat. Camera stays perfectly still on a wide establishing shot. Smooth, continuous transition. Duration: 7 seconds.

Flowers shift from summer blooms (roses, lavender) to autumn colours (chrysanthemums, dahlias). The lawn subtly shifts from lush green to a warmer tone. Camera stays perfectly still. ## Tips for Best Results

1. **Background matching**: If your website has a dark background, change "white background" to "black background" in all prompts
2. **Consistency**: Always use the first generated image as reference for the second
3. **Quality**: Always use 2K (2048px) minimum. 1K looks terrible on desktop
4. **Iterations**: Generate 4 versions and pick the best. First try rarely perfect
5. **Aspect ratio**: 16:9 for landscape/desktop. Consider 9:16 for a mobile-specific version
6. **Enhance OFF**: Never use the enhance/upscale prompt option in Kling — it changes the style
7. **Duration**: 5-7 seconds is optimal. Longer = more frames = slower loading
8. **Motion**: Less is more. Subtle, elegant movement beats dramatic action
